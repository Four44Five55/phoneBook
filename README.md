# PhoneBook - Телефонный справочник

Простой телефонный справочник на JavaFX с возможностью импорта/экспорта данных в CSV.

## Возможности

- Добавление, редактирование, удаление контактов
- Поиск по ФИО, телефону, отделу, примечаниям (поиск игнорирует регистр и считает ё=е)
- Импорт данных из CSV файла
- Экспорт данных в CSV файл (совместим с Excel)
- Переключение тёмной/светлой темы
- Сохранение размера окна и ширины столбцов

## Поля контакта

- **ФИО** - полное имя
- **Отдел** - подразделение
- **Телефон** - номер телефона (автоформатирование в формат 8-XXX-XXX-XX-XX)
- **Примечание** - дополнительная информация

## Формат CSV

```
ФИО;Отдел;Телефон;Примечание
Иванов Иван;IT;8-999-123-45-67;разработчик
Петров Петр;HR;8-910-111-22-33;менеджер
```

## Требования

- Java 11 или выше (для портативной версии не требуется)

## Сборка из исходников

```bash
./gradlew build
```

## Создание портативной версии

```bash
./gradlew portable
```

Готовое приложение будет в папке `build/PhoneBook/`

## Запуск

**Из исходников:**
```bash
./gradlew run
```

**Портативная версия:**
Просто запустите `PhoneBook.exe` - Java не требуется.

## База данных

Приложение использует SQLite. Файл базы данных: `db/phonebook.db`
